Step1: Assumptions

For proportion

  1. The variable measured is categorical.
  2. The data are obtained using randomization.
  3. The sample size n is sufficiently large that the sampling distribution of the sample proportion $\hat{p}$ is approximately Normal when the null is true.

(This will be true if n⋅p0⋅(1 − p0) ≥ 5, where p0 is the value specified in H0.

For mean

  1. The variable measured is quantitative.
  2. The data are obtained using randomization.
  3. The population distribution is approximately Normal. This assumption is crucial when n is small.

Step2: Stating hypothesis

The null hypothesis of a test about the mean has the form:

H₀: μ = μ₀ where μ₀ is the hypothesized mean of the population.

A two-sided alternative hypothesis would be:

H₁: μ ≠ μ₀

A one-sided alternative hypothesis is either:

H₁: μ < μ₀ or H₁: μ > μ₀

Step3: Test statistics and its null distribution

For proportion

The test statistic of a test describes how far that point estimate falls from the parameter value given in the null hypothesis.

The test statistic is the distance of (\hat{p}) from (p_0) in terms of its standard deviation:

[ Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1 - p_0)}{n}}} ]

This (Z \sim N(0, 1)).
