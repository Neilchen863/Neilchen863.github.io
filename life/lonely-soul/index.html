Loneliness and meaningfulness

2024-10-19

总是孤独-> 开始玩桌游->有了朋友 -> 开心 -> 独处时更加感到孤独 -> 质疑意义 -> 玩桌游到底有没有意义 -> 或许只是为了缓解孤独？ -> 思考桌游的意义 -> 锻炼脑力口才 -> 只是借口 -> 或许逃避孤本身也是一种意义 -> 人们倾向于为自己所做的事赋予一个崇高的意义 -> 但这种意义很容易被解构 -> 因为本质是虚无 -> 放弃这种无谓的意义感 -> 正视自己的欲望与孤独 -> 也许这才是真正的意义

下午4点58分，我走下拥挤的D2校车，拨开人群逆着人流走向教室，突然觉得很幸福。

此时的我刚从一场酣睡中醒来，虽说睡得无比香甜，但作为这40个小时里唯一合眼的4个小时，又被闹钟强行拉起来上课，大脑依旧混沌，且心情也免不了有些糟糕。热带下午的阳光依旧炽烈， 我五指并拢靠在眼睛上方，试图遮挡住那些刺眼的光芒。可这却完全是一场徒劳，阳光大片大片地洒下，惊醒了这片大地上每一个还在沉睡的人。我完全清醒了，首先想起的却是刚才做的那场大梦。

可是这真的是一场梦境吗？手机里的照片仍然静静地躺在相册里，通宵的痕迹又如此明显地呈现在我的面容，但同时这一切却又显得如此恍惚而不真实。作为这学期的第一场真正意义的通宵， 没人知道它为何而起，也没人知道它何以延续，但它真真切切地发生了，并让我多多少少地察觉
